## Visão Geral
Gestão de escala de serviço e gerar um documento ao final para ser enviado.
A aplicação junta as fucnionalidades de gerenciamneto de contato, podendo com um clique acionar uma pessoa para conversar no whatsapp.

## Funcionalidades

Abaixo as principais funcionalidades ou características do projeto.

* CRUD de Servidores;
* Visualziar uma lista de servidores;
* Cadastrar uma escala de serviço com o servidores disponíveis;
* Facildiade de troca de serviço dos servidores;
* Gerar a escala automaticaente.
* Link do whatsapp para mensagem rápida.
* Gerar um documento com a escala de serviço.

## Como Usar

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/denisiorodrigues/Adopet.git
    ```
2.  **Navegue até a pasta do projeto**
    ```bash
    cd EscalaDeServico/
    ```
3. **Execute a aplicação Web:**

    ```bash
    dotnet run
    ```
4. **Subindo a base de dados:**

    ```bash
    docker compose mysql up -d
    ```
